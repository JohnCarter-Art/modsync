# ModSync — Автоматическое обновление модов Sims 4

Следит за установленными модами для **The Sims 4** и автоматически проверяет
наличие обновлений. Помогает держать коллекцию модов в актуальном состоянии,
чтобы игра работала стабильно после выхода патчей от EA.

## О проекте

Игроки в The Sims 4 часто используют десятки и сотни сторонних модов.
После каждого обновления игры часть модов перестаёт работать — разработчики
выпускают новые версии, но отслеживать это вручную утомительно и легко
пропустить.

**ModSync** решает эту проблему:

- Сканирует установленные моды.
- Определяет, какие из них устарели.
- Проверяет доступные обновления (через CurseForge, ModDB, GitHub Releases
  и другие источники).
- Уведомляет пользователя о найденных обновлениях.

## Установка

> **Требования:** Python 3.11+

```bash
# Клонирование репозитория
git clone https://github.com/JohnCarter-Art/modsync.git
cd modsync

# (Опционально) Создание и активация виртуального окружения
python3 -m venv venv
source venv/bin/activate     # Linux / macOS
# venv\Scripts\activate      # Windows

# Установка зависимостей
pip install -r requirements.txt
```

## Использование

```bash
python -m modsync --help
```

(Пример использования будет дополнен по мере разработки.)

## Конфигурация

Планируется поддержка файла `modsync.json` в домашней директории пользователя
или рядом с исполняемым файлом. Пример:

```json
{
  "sims4_mods_path": "~/Documents/Electronic Arts/The Sims 4/Mods",
  "check_interval_hours": 24,
  "sources": ["curseforge", "github"],
  "ignored_mods": ["SomeSpecificMod.package"]
}
```
